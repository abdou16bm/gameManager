this file is for window Application configuration :


1 - size : (this option will be deactivated when the application is in fullscreen)

   a) - 1 -> SD (800x600)
 
   b) - 1 -> HD (1280x720)

   c) - 3 -> FULL HD (1920x1080)

////////////////////////////////

2 - full :

   a) - 0 -> not fullscreen

   b) - 1 -> fullscreen

///////////////////////////////

3 - top :

   a) - 0 -> App not in foreground

   b) - 1 -> App in foreground
